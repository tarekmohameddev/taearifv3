# صفحة إضافة عقد إيجار جديد

## الوصف
صفحة منفصلة لإضافة عقود الإيجار الجديدة في المسار `/dashboard/rental-management/create`.

## الميزات
- نموذج شامل لإدخال بيانات المستأجر
- اختيار المشروع والعقار من قوائم منسدلة
- التحقق من صحة البيانات
- تصميم متجاوب مع اللغة العربية
- إعادة التوجيه التلقائي بعد الإضافة

## الملفات
- `page.tsx` - الصفحة الرئيسية
- `layout.tsx` - تخطيط الصفحة مع الشريط الجانبي والرأس

## الاستخدام
يمكن الوصول للصفحة من خلال:
1. زر "إضافة إيجار جديد" في صفحة إدارة الإيجارات
2. الرابط المباشر: `/dashboard/rental-management/create`
